## ✨ Features

- 📊 **Dashboard** – Get a bird’s‑eye view of your day.
- ✅ **Daily Routine** – Track and complete your daily rituals.
- 🍅 **Pomodoro Timer** – Stay focused with smart timing suggestions based on your energy level.
- 🎯 **MIT Planner** – Identify and crush your Most Important Tasks.
- ⚡ **Energy Input** – Log your energy levels and get personalised productivity tips.
- 🌱 **Growth Module** – Set learning goals and track your progress.
- 📓 **Reflection Journal** – Capture thoughts, wins, and lessons learned.
- 🔁 **Habit Tracker** – Build and maintain positive habits.
- 📅 **Weekly Review** – Reflect on the past week and plan the next one.
- 📈 **Analytics Dashboard** – Visualise your productivity trends over time.
- 🎵 **Music Player** – Listen to your favourite focus playlists without leaving the app.
- ⏱️ **Task Timer with Session History** – Time your tasks and review past sessions.
- 🎨 **Fully Responsive** – Works seamlessly on desktop, tablet, and mobile.
- 🌙 **Dark Mode** – Easy on the eyes, day or night.

## 🚀 Quick Start

Get your own copy up and running in minutes.

### Prerequisites

- **Node.js** (v20 or later)
- **npm** or **yarn** or **pnpm**

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/PsymoNiko/life-planner.git
   cd life-planner
```

1. Install dependencies
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```
2. Start the development server
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```
3. Open your browser and visit http://localhost:3000. Start planning your life! 🎉
